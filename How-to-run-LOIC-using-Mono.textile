**Linux (Ubuntu / Debian / Fedora)**

* Install monodevelop from: [[https://www.monodevelop.com/download/#fndtn-download-lin]]
* Shell script for legacy distributions with mono .net 4.0: [[https://github.com/NewEraCracker/LOIC/blob/master/loic.sh]]
* Shell script for distributions with mono .net 4.5: [[https://github.com/NewEraCracker/LOIC/blob/master/loic-net4.5.sh]]

**Mac OS X (Old instructions - Not tested on latest versions)**

# Download LOIC (with windows binaries) from download page and extract in desktop.
# Download mono from: [[https://www.mono-project.com/download/stable/]]
# Install mono.
# Open terminal and type
_mono /path_to_loic/debug/LOIC.exe_
# LOIC should open (check tray too)
